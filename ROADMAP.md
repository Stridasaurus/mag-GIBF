## 1. Purpose

Sequence the experiments that retire the two critical unknowns about SECS‚ÄĎGIBF ‚ÄĒ **whether realistic FLR inter‚ÄĎsource phase survives to the ground CSM** (Experiment A) and **whether per‚ÄĎmode independence earns its keep against a joint shared‚ÄĎsupport inversion** (Experiment B ‚Üí D2) ‚ÄĒ so that the full‚ÄĎpipeline direction is chosen from evidence, not from the method's name.

CF identifiability is **no longer an unknown**. In this model class it is settled negatively by theorem: for radial FACs ‚ÄĒ which the SECS CF basis has *by construction* ‚ÄĒ the CF current plus its FAC produces exactly zero ground magnetic field, independent of the conductance distribution (Fukushima 1976; Amm 1997). It is carried as **Validation Gate V** (does the code implement the theorem?), cited in the paper as theory, and never re‚ÄĎlitigated as an experiment.

## 2. Invariant vision & boundaries

True across *every* branch below. Anything that varies by branch lives in a stub, not here.

**Vision.** Physically consistent imaging of ionospheric current systems from magnetometer arrays, using a sparse SECS source basis, with every methodological claim (does FLR phase survive to the ground CSM? does per‚ÄĎmode independence earn its keep, and in which ŌÜ / misspecification regime?) decided by a pre‚ÄĎregistered, executable rule rather than asserted ‚ÄĒ and with settled theory (Fukushima/Amm) cited as theory, not re‚ÄĎrun as experiment.

**Domain of validity (carried into every manifesto).** The quasi‚ÄĎstatic / real‚ÄĎkernel treatment holds when array aperture `L` and signal period `T` satisfy `L/c ‚Č™ T` (equivalently `L ‚Č™ őĽ`): ground arrays of 10¬≤‚Äď10¬≥ km, frequencies 10‚ĀĽ¬≥‚Äď1 Hz (Pc3‚Äď5, Pi2, substorm current wedge, FAC structures; IMAGE / SuperMAG‚ÄĎclass arrays). Full‚ÄĎwave regimes (e.g. VLF > 3 kHz) are **out of scope** and would require a dyadic EM Green's function.

**Invariants ‚ÄĒ never violate, in any branch:**

- **Never** apply a propagation phase to the transfer matrix. `A` is **real**; there is no `exp(i k r)`. The *only* exception anywhere in the program is the deliberate phantom‚ÄĎphase ablation (Experiment B4). Outside it, `assert np.isrealobj(A)`.
- **Never** include CF columns in `A` for any ground inversion ‚ÄĒ excluded by theorem, not by threshold. The only CF computation anywhere in the program is Gate V's probe.
- **Always** build the CSM from **complex frequency‚ÄĎbin phasor snapshots**, never raw real time samples. Real `X` ‚áí real `S` ‚áí real eigenvectors ‚áí the premise collapses. All phase lives in the complex source phasors ‚Üí complex CSM ‚Üí complex eigenvectors.
- **Always** carry the eigenvalue magnitude: the eigenmode is `vŠĶĘ = ‚ąöőĽŠĶĘ ¬∑ uŠĶĘ` (Suzuki Eq. 3). Never invert the unit eigenvector.
- **Always** feed any head‚ÄĎto‚ÄĎhead method comparison the **identical** `V` and the **identical** real `A`. Identical inputs are *necessary but not sufficient*: GIBF (per‚ÄĎmode L1‚ÄĎIRLS + grid reduction) and MMV‚ÄĎL1 (joint L2,1) cannot share a literal parameter set, so the comparison is only fair under an explicit **matched‚ÄĎregularization / matched‚ÄĎeffective‚ÄĎsparsity protocol**, with grid reduction applied **symmetrically** (both methods or neither), and with the **sign of the gap reported across a band of regularization strengths** at the primary cells (a verdict robust to the tuning knob is a result; a verdict at one knob setting is an anecdote). The protocol is pinned in the manifesto; the *requirement* is invariant here.
- **Always** report the **signed** `GIBF‚ąíMMV` gap, never a magnitude alone. The thesis is sign‚ÄĎneutral (¬ß7 Stub 0); either sign is the finding.
- **Always** pre‚ÄĎregister each experiment's decision rule and threshold *before* looking at results, and record them in `DECISION_MEMO.md`.
- **Log hygiene:** corrections to ¬ß8 are appended as new dated entries referencing the old entry ‚ÄĒ never inline edits or annotations. (The 2026‚ÄĎ07‚ÄĎ01 annotation inside the 2026‚ÄĎ06‚ÄĎ30 pre‚ÄĎregistration entry is grandfathered as the last exception.)

## 3. Glossary (cross‚ÄĎbranch)

One name per concept, used identically in this roadmap, in every manifesto, and in code.

| Term | Meaning |
|---|---|
| **SECS** | Spherical Elementary Current Systems; the source basis. |
| **CF** | Curl‚ÄĎfree SECS part ‚ÄĒ divergence of the ionospheric current, closing via FACs. Excluded from all ground inversions by theorem (see **Fukushima theorem**, **Gate V**). |
| **DF** | Divergence‚ÄĎfree SECS part ‚ÄĒ rotational/solenoidal current closed within the sheet. The sole basis for ground `A`. |
| **FAC** | Field‚ÄĎaligned current. |
| **Fukushima theorem** | Radial FACs + their CF current produce **exactly zero** ground magnetic field below the ionosphere, for **any** conductance distribution (Fukushima 1976 proved the uniform case; Amm 1997 removed the uniformity assumption). The crucial assumption is *radial* FAC ‚ÄĒ which the SECS CF elementary system satisfies by construction, at every pole latitude. Hence CF **is** (not "may be") unrecoverable from ground data in this model class; the question is a theorem, not an experiment. Lifting it requires non‚ÄĎradial FACs (see **Stub X‚Ä≤**). |
| **`A` (transfer matrix)** | Real geometric coupling matrix `B(ŌČ) = A ¬∑ s(ŌČ)`, mapping unit SECS amplitudes ‚Üí ground field. Real in every branch except the B4 ablation. **DF columns only**; CF columns exist only inside Gate V's probe. |
| **`secsy`** | Library providing the SECS field kernels `(Ge, Gn, Gu)`. API has drifted across versions; pin and verify per build. |
| **CSM** | Cross‚ÄĎspectral matrix `S = (1/N) X XŠīī`; complex Hermitian PSD. |
| **eigenmode `vŠĶĘ`** | `‚ąöőĽŠĶĘ ¬∑ uŠĶĘ` (Suzuki Eq. 3). The shared input to every inversion under comparison. |
| **GIBF** | Generalized Inverse Beamforming: invert each eigenmode **independently** with a per‚ÄĎmode sparse (L1‚ÄĎIRLS) prior, then accumulate intensity across modes. The **per‚ÄĎmode independence** under test (colloquially "the eigen layer" ‚ÄĒ but note both GIBF and MMV‚ÄĎL1 consume the same eigendecomposition; D2 adjudicates per‚ÄĎmode vs shared‚ÄĎsupport, not eigendecomposition vs none). |
| **MMV‚ÄĎL1** | The competitor: a single **joint** inversion of the same `K` eigenmodes with a **shared‚ÄĎsupport** (L2,1 row‚ÄĎsparse) prior. Same `V`, same `A`; differs from GIBF only in shared‚ÄĎsupport vs independent‚ÄĎper‚ÄĎmode. |
| **IRLS** | Iteratively reweighted least squares; the L1 solver core. |
| **grid reduction (ő≤)** | Irreversible pruning of the active source set by factor ő≤ = 0.9 per iteration; switches the solve underdetermined ‚Üí overdetermined. Applied symmetrically (both methods or neither); report the method √ó reduction 2√ó2 where feasible. |
| **mode selection** | Choosing the number of signal eigenmodes via complex AIC / MDL (Wax‚ÄďKailath 1985). In **B6a** the *estimated* `KŐā` is fed to the solvers (not oracle `K`). |
| **observability ratio `ŌĀ_obs(j)`** | `‚ÄĖA¬∑e_CF(j)‚ÄĖ / ‚ÄĖA¬∑e_DF(j)‚ÄĖ` at grid point `j`. **Repurposed 2026‚ÄĎ07‚ÄĎ01:** now Gate V's validation statistic ‚ÄĒ expected ‚Čą machine precision when the adapter implements the full Fukushima pair; a materially nonzero value indicates an **adapter bug** (most likely: horizontal CF current returned without its FAC), never physics. |
| **`‚ÄĖIm S‚ÄĖ/‚ÄĖS‚ÄĖ`** | CSM imaginary fraction (rotation‚ÄĎinvariant). Card A's **co‚ÄĎprimary** statistic; the observable online gate for D2's liability branch. Finite‚ÄĎN floor ~`1/‚ąöN` (measured 0.060 at N=64, 0.0013 at N=10‚ĀĶ). |
| **`P_sep`** | Separation success rate: fraction of trials in which both true sources resolve as distinct, correctly‚ÄĎplaced peaks. **Pinned (¬ß8‚ÄĎv):** *correctly placed* = within **1 grid cell** of a true source under minimum‚ÄĎcost matched assignment; *distinct* = the two reported peaks match **different** true sources. |
| **inter‚ÄĎsource relative phase `ŌÜ`** | The phase of the complex coherence `ŌĀ = |ŌĀ|¬∑e^{iŌÜ}` between two coherent sources. **Load‚ÄĎbearing:** with a real `A`, the CSM acquires exploitable imaginary structure *only* when `ŌÜ ‚Č† 0, 180¬į`. Verified: residual eigenvector complexity ‚Čą 0.00 at ŌÜ=0 and 180¬į, 0.22/0.44/0.65 at 30/60/90¬į, ‚Čą 0.14 incoherent at N=64 (finite‚ÄĎsample). |
| **residual eigenvector complexity `őļ`** | Global‚ÄĎphase‚ÄĎremoved imaginary content of the top‚ÄĎ2 CSM eigenmodes (`min_őł ‚ÄĖIm(e^{iőł}u)‚ÄĖ/‚ÄĖu‚ÄĖ`). Reported at the **source** (`őļ_source`, from `R_s`) and at the **ground** (`őļ_ground`, from the ground CSM after the real kernel); the gap is the ionospheric‚ÄĎintegration "washout." **Caveat:** eigenvector rotation is ill‚ÄĎconditioned when `őĽ‚āĀ ‚Čą őĽ‚āā` (equal‚ÄĎpower sources) ‚ÄĒ always report the eigenvalue gap alongside, Monte‚ÄĎCarlo‚ÄĎaverage in Tier 2, and cross‚ÄĎcheck with the rotation‚ÄĎinvariant `‚ÄĖIm S‚ÄĖ/‚ÄĖS‚ÄĖ`. |
| **FLR quality factor `Q`** | `Q ‚Č° ŌČ/ő≥` for the driven‚ÄĎdamped‚ÄĎoscillator field‚ÄĎline‚ÄĎresonance model (Card A); realistic `Q ‚Čą 5‚Äď20` sets how narrow (in latitude) the ~180¬į source cross‚ÄĎphase transition is. |
| **`őĒrŐĄ`** | Mean two‚ÄĎsource localisation error (minimum‚ÄĎcost matched assignment; error averaged over matched pairs). |
| **phantom‚ÄĎphase ablation** | Experiment B4 only: a *fake* complex `A_phase = A ‚äô exp(i¬∑ő¶)` emulating a propagating‚ÄĎwave phase ramp, run GIBF‚ÄĎonly, to quantify what the absent acoustic phase diversity *would* buy. |
| **B6 (stress cells)** | Prior‚ÄĎmisspecification cells: (a) solver‚ÄĎfed `KŐā` from AIC/MDL, (b) ‚Č§1‚ÄĎcell source‚ÄĎposition jitter across the snapshot window. | **Gate V** | The kernel validation gate (was Experiment A): the `secsy` CF‚ÄĎpair probe + realness assertions + DF spot checks. Not a decision; a pass/halt. |
| **snapshot** | One complex frequency‚ÄĎbin phasor column of `X`. |

## 4. Research‚ÄĎphase exit conditions

These are *not* the project's success criteria ‚ÄĒ they are the criteria for **stopping deciding and locking the next manifesto**. The viability test (Manifesto #1) is the effort that reaches them.

Research is "done deciding" when **all** of the following hold:

1. **Correctness gates green** so the decisions are trustworthy: single‚ÄĎsource exact recovery for all three solvers in the noise‚ÄĎfree well‚ÄĎseparated case; CSM provably *not* near‚ÄĎreal for a multi‚ÄĎphase scenario; ‚ąöőĽ carried (Suzuki Eq. 3); the `secsy` adapter's verified version + conventions recorded; **Gate V passed** ‚ÄĒ CF + FAC ground field ‚Č° 0 at machine precision at every latitude preset, documented as the theorem's numerical confirmation (headline calibration, not a finding). `pytest -q` green in < 1 minute.
2. **Q1 (Experiment A) resolved:** the H‚ÄĎA verdict landed per the ¬ß5 node‚ÄĎA bands (holds / marginal / fails), with the őļ_source‚ÄĎvs‚ÄĎőļ_ground washout gap reported; B2's ŌÜ range, weighting, and `|ŌĀ|` range set from it; the paper's framing fork chosen.
3. **Q2 (Experiment B) resolved:** B1 complete with mean ¬Ī 95 % CI per method per cell; the ¬ß[D2] rule applied ‚ÄĒ **both the win rule and the loss rule (¬ß8‚ÄĎvi)** ‚ÄĒ to land the D2 outcome; B2 (coherent, ŌÜ‚ÄĎswept), B3 (mode selection), B4 (phantom‚ÄĎphase), and **B6 (misspecification stress cells, ¬ß8‚ÄĎvii)** complete. **B2 may not be designed until Experiment A has resolved** ‚ÄĒ its result sets B2's realistic ŌÜ range/weighting and `|ŌĀ|`, and the primary confirmatory cells must be pre‚ÄĎregistered before B2 runs (¬ß8‚ÄĎiii).
4. **`DECISION_MEMO.md` filled** with thresholds pre‚ÄĎregistered before results, all decisions recorded, and a recommendation written.
5. **Methods paper drafted** to submission‚ÄĎready state for the target venue. *Confirmed synthetic‚ÄĎonly*: no real‚ÄĎdata example is required, so the real‚ÄĎdata demo stays downstream (Stub F/S) and is **out of Manifesto #1's scope**. Much of the paper ‚ÄĒ intro, physics, domain of validity, the Fukushima/Gate‚ÄĎV paragraph, algorithm, the pre‚ÄĎregistered hypotheses and decision rules ‚ÄĒ drafts from the manifesto and planning docs *in parallel with the build*; only the results/decision sections wait on the experiments.

**No‚ÄĎkill floor (quadrant map).** Every cell of (Experiment A outcome) √ó (D2 outcome) carries a headline:

- *H‚ÄĎA holds √ó GIBF wins:* per‚ÄĎmode imaging earns its keep in the physically dominant ULF regime, with B4 as mechanism.
- *H‚ÄĎA holds √ó GIBF loses:* exploitable phase exists, survives to the ground ‚ÄĒ and per‚ÄĎmode inversion *still* cannot beat a joint inversion of the same modes; B4 shows why (acoustic GIBF's power came from kernel phase diversity, which no quasi‚ÄĎstatic real‚ÄĎkernel problem has). A transferable negative for the whole real‚ÄĎkernel inverse‚ÄĎproblem class.
- *H‚ÄĎA fails (either mode):* Experiment A itself is the headline ‚ÄĒ FLR cross‚ÄĎphase, robustly measured *between stations*, does not survive into the ground CSM's eigenstructure (or is near‚ÄĎreal at the source). A ground‚ÄĎobservability result the ULF community can use independent of GIBF.
- *B6 win with clean‚ÄĎcell loss:* the method's actual value proposition is robustness to prior misspecification, not resolution ‚ÄĒ arguably the most useful positive available.

The research phase cannot terminate in project death; it terminates by *selecting which downstream manifesto to write*.

When met: select the downstream branch by the `(Q1 √ó Q2)` outcome (¬ß7) and expand that stub into Manifesto #2.
